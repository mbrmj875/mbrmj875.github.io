الأجزاء الرئيسية للمشروع

الجزء 1: تثبيت نظام التشغيل

تثبيت Ubuntu Server 20.04 LTS بدون واجهة رسومية والتحقق من الإصدار

$ cat /etc/issue
# التحقق من إصدار Ubuntu
الجزء 2: إدارة المستخدمين

إنشاء مستخدمين جديدين وإضافتهم إلى مجموعات إدارية

$ sudo adduser baqer-1
$ sudo usermod -aG adm baqer-1
$ cat /etc/passwd
الجزء 3: تكوين الشبكة

إعداد اسم المضيف، المنطقة الزمنية، وواجهات الشبكة

$ sudo hostnamectl set-hostname user-1
$ sudo timedatectl set-timezone Europe/Moscow
$ ip link show
$ ping 1.1.1.1
الجزء 4: تحديثات النظام

تحديث قائمة الحزم وترقية النظام

$ sudo apt update
$ sudo apt upgrade
الجزء 5: صلاحيات Sudo

منح صلاحيات إدارية للمستخدمين وتغيير اسم المضيف

$ sudo usermod -aG sudo username
$ sudo hostnamectl set-hostname new-hostname
الجزء 6: خدمة الوقت

إعداد مزامنة الوقت التلقائية عبر NTP

$ sudo apt install ntp -y
$ timedatectl show
الجزء 7: محررات النصوص

تثبيت واستخدام VIM، NANO، وMCEDIT

$ sudo apt install vim nano joe mc -y
$ vim test_VIM.txt
# i للإدراج، ESC للأوامر، :wq للحفظ والخروج
الجزء 8: خدمة SSH

إعداد OpenSSH مع منفذ مخصص وبدء تلقائي

$ sudo apt install openssh-server -y
$ sudo systemctl enable ssh
$ netstat -tan | grep 2022
الجزء 9: مراقبة النظام

استخدام top و htop لمراقبة العمليات والموارد

$ sudo apt install htop
$ top
$ htop
الجزء 10: إدارة الأقراص

تحليل أقسام القرص باستخدام fdisk

$ sudo fdisk -l
# عرض معلومات القرص والأقسام
الجزء 11: تحليل استخدام نظام الملفات

مراقبة مساحة القرص باستخدام df

$ df -Th /
# عرض استخدام المساحة بصيغة مقروءة
الجزء 12: تحليل مساحة المجلدات

تتبع استهلاك التخزين باستخدام du

$ du -sh /home
$ du -sh /var/log
الجزء 13: التحليل التفاعلي لاستخدام القرص

استخدام ncdu للتحليل البصري للتخزين

$ sudo apt install ncdu -y
$ ncdu /home
الجزء 14: تحليل سجلات النظام

مراقبة وإدارة سجلات النظام والأمان

$ less /var/log/syslog
$ grep "sshd" /var/log/auth.log
الجزء 15: جدولة المهام

إدارة المهام الآلية باستخدام Cron

$ crontab -e
# */2 * * * * /usr/bin/uptime
$ crontab -l

أمثلة التكوين المتقدمة

تكوين شبكة ثابتة:
# /etc/netplan/00-installer-config.yaml
network:
  version: 2
  ethernets:
    ens33:
      dhcp4: no
      addresses: [192.168.1.100/24]
      gateway4: 192.168.1.1
      nameservers:
        addresses: [8.8.8.8, 1.1.1.1]
  renderer: networkd
تكوين SSH آمن:
# /etc/ssh/sshd_config
Port 2022
PermitRootLogin no
PasswordAuthentication yes
X11Forwarding no
MaxAuthTries 3
ClientAliveInterval 300
ClientAliveCountMax 2

مراقبة النظام واستكشاف الأخطاء

🔍 مراقبة العمليات
$ ps aux | grep ssh
$ top -p 1234
$ htop -u username
📊 مراقبة الموارد
$ free -h
$ df -h
$ iostat -x 1
تحليل السجلات:
$ journalctl -u ssh -f
$ tail -f /var/log/syslog
$ grep -i error /var/log/syslog
فحص الشبكة:
$ ss -tuln
$ netstat -tulpn
$ ip addr show
$ ping -c 4 google.com
