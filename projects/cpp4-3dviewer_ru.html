Продвинутое приложение для просмотра 3D wireframe моделей с поддержкой паттернов проектирования и современного GUI

Паттерны проектирования 3D графика GUI разработка ООП

Обзор проекта

Проект 3DViewer v2.0 представляет собой усовершенствованную версию приложения для просмотра 3D wireframe моделей, разработанную с использованием современных паттернов проектирования и архитектурных принципов. Приложение поддерживает загрузку моделей из OBJ файлов и выполнение аффинных преобразований.

Архитектура MVC

🎯 Принципы разделения ответственности
Компонент Ответственность Запрещено
Model Бизнес-логика, данные, преобразования Код интерфейса
View Отображение, обработка UI событий Бизнес-логика
Controller Тонкий слой управления, модификация модели Логика интерфейса и бизнеса

class ModelController {
