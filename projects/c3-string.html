نظرة عامة على المشروع

مشروع C3_s21_stringplus هو تنفيذ كامل لمكتبة معالجة النصوص في لغة C، حيث يقوم بإعادة بناء جميع دوال مكتبة string.h القياسية مع إضافة وظائف متقدمة مستوحاة من لغة C#. يهدف المشروع إلى فهم عميق لكيفية عمل دوال النصوص على مستوى الذاكرة وإدارة الموارد.

  • اختبارات شاملة لكل الدوال
  • توثيق مفصل للكود

أمثلة على الاستخدام

مثال 1: تحويل حالة النصوص
#include "s21_string.h"

int main() {
    char *text = "Hello World";
    char *upper = to_upper(text);
    char *lower = to_lower(text);

    printf("Original: %s\n", text);
    printf("Upper: %s\n", upper);
    printf("Lower: %s\n", lower);

    free(upper);
    free(lower);
    return 0;
}
مثال 2: إدراج وتقطيع النصوص
char *result = insert("Hello", " Beautiful", 5);
char *trimmed = trim(" Hello World ", " ");

printf("Inserted: %s\n", result); // "Hello Beautiful"
printf("Trimmed: '%s'\n", trimmed); // "Hello World"
