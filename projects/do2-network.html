مخطط عنونة IP:
  • الشبكة 10.10.0.0/18: الشبكة الداخلية 1
  • الشبكة 10.100.0.0/16: اتصال بين الموجهات
  • الشبكة 10.20.0.0/26: الشبكة الداخلية 2
  • نطاق DHCP: 10.20.0.2 - 10.20.0.50

أمثلة التكوين

تكوين Netplan:
# /etc/netplan/00-installer-config.yaml
network:
  version: 2
  ethernets:
    eth0:
      addresses: [10.10.0.2/18]
      routes:
        - to: 0.0.0.0/0
          via: 10.10.0.1
      nameservers:
        addresses: [8.8.8.8]
جدار ناري بـ iptables:
#!/bin/sh
# مسح جميع القواعد
iptables -F
iptables -t nat -F
iptables -X

# السياسات الافتراضية
iptables -P INPUT DROP
iptables -P FORWARD DROP
iptables -P OUTPUT ACCEPT

# السماح بالخدمات المحددة
iptables -A INPUT -p tcp --dport 22 -j ACCEPT
iptables -A INPUT -p tcp --dport 80 -j ACCEPT
iptables -A INPUT -p icmp -j ACCEPT
