Обзор проекта

Проект C7_BrickGame_v1.0 представляет собой полную реализацию классической игры Tetris на языке C с использованием библиотеки NCurses для терминального интерфейса. Проект отличается продвинутой модульной архитектурой, разделяющей игровую логику и пользовательский интерфейс, с реализацией конечного автомата для управления потоком игры.

Система очков и уровней

Продвинутый алгоритм подсчета очков
// Система расчета очков void update_score(GameState_t *gs, int lines_cleared) {   switch (lines_cleared) {     case 1: gs->score += 100; break;     case 2: gs->score += 300; break;     case 3: gs->score += 700; break;     case 4: gs->score += 1500; break;   }   // Обновление уровня каждые 600 очков   gs->level = (gs->score / 600) + 1;   gs->speed = 1000 - (gs->level * 100); }
