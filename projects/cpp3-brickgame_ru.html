Обзор проекта

Проект BrickGame Snake представляет собой реализацию классической игры "Змейка" с использованием современных архитектурных паттернов проектирования. Проект демонстрирует практическое применение MVC, MVP и MVVM паттернов в разработке игровых приложений на C++.

Конечные автоматы (Finite State Machines)

🎯 Состояния игры
Состояние Описание Действия
START Начальное состояние Инициализация игры
MOVING Движение змейки Обновление позиции, проверка столкновений
PAUSED Пауза Приостановка таймера
GAME_OVER Конец игры Отображение результата
WIN Победа Поздравление игрока
Пример реализации FSM
class GameStateMachine {
private:   GameState current_state_ = GameState::START; public:   void handleEvent(GameEvent event) {     switch (current_state_) {       case GameState::START:         if (event == GameEvent::START_GAME) {           current_state_ = GameState::MOVING;           initializeGame();         }         break;       case GameState::MOVING:         if (event == GameEvent::PAUSE) {           current_state_ = GameState::PAUSED;         } else if (checkCollision()) {           current_state_ = GameState::GAME_OVER;         }         break;     }   } };

Исторический контекст

Игра Snake имеет богатую историю, начиная с 1977 года и версии от Nokia 6110 (1997). Архитектурные паттерны MVC, MVP и MVVM были разработаны для разделения ответственности в приложениях.

"The main need for this pattern is related to the developers' desire to separate the program's business logic from the views" - Trygve Reenskaug, создатель MVC (1978)
