Project Overview

The C3_s21_stringplus project is a complete implementation of a text processing library in C language, rebuilding all standard functions of the string.h library with the addition of advanced functions inspired by C# language. The project aims to provide a deep understanding of how text functions work at the memory level and resource management.

Usage Examples

Text Case Conversion Example:
// Example of using to_upper and to_lower functions
#include "s21_string.h"
#include <stdio.h>
#include <stdlib.h>

int main() {
    char *text = "Hello World";
    char *upper = to_upper(text);
    char *lower = to_lower(text);

    printf("Original: %s\n", text);
    printf("Upper: %s\n", upper);
    printf("Lower: %s\n", lower);

    free(upper);
    free(lower);
    return 0;
}
Text Insertion and Trimming Example:
// Example of using insert and trim functions
char *result = insert("Hello", " Beautiful", 5);
char *trimmed = trim(" Hello World ", " ");

printf("Inserted: %s\n", result); // Output: "Hello Beautiful"
printf("Trimmed: '%s'\n", trimmed); // Output: "Hello World"

free(result);
free(trimmed);
