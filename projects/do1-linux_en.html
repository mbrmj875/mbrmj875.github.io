Main Project Parts

Part 1: Operating System Installation

Installing Ubuntu Server 20.04 LTS without GUI and verifying version

$ cat /etc/issue
# Checking Ubuntu version
Part 2: User Management

Creating new users and adding them to administrative groups

$ sudo adduser baqer-1
$ sudo usermod -aG adm baqer-1
$ cat /etc/passwd
Part 3: Network Configuration

Configuring hostname, timezone, and network interfaces

$ sudo hostnamectl set-hostname user-1
$ sudo timedatectl set-timezone Europe/Moscow
$ ip link show
$ ping 1.1.1.1
Part 4: System Updates

Updating package lists and upgrading the system

$ sudo apt update
$ sudo apt upgrade
Part 5: Sudo Privileges

Granting administrative privileges to users and changing hostname

$ sudo usermod -aG sudo username
$ sudo hostnamectl set-hostname new-hostname
Part 6: Time Service

Setting up automatic time synchronization via NTP

$ sudo apt install ntp -y
$ timedatectl show
Part 7: Text Editors

Installing and using VIM, NANO, and MCEDIT

$ sudo apt install vim nano joe mc -y
$ vim test_VIM.txt
# i for insert, ESC for commands, :wq to save and exit
Part 8: SSH Service

Setting up OpenSSH with custom port and auto-start

$ sudo apt install openssh-server -y
$ sudo systemctl enable ssh
$ netstat -tan | grep 2022
Part 9: System Monitoring

Using top and htop for process and resource monitoring

$ sudo apt install htop
$ top
$ htop
Part 10: Disk Management

Analyzing disk partitions with fdisk

$ sudo fdisk -l
# Displaying disk and partition information
Part 11: Filesystem Usage Analysis

Monitoring disk space usage with df

$ df -Th /
# Displaying space usage in human-readable format
Part 12: Directory Space Analysis

Tracking storage consumption with du

$ du -sh /home
$ du -sh /var/log
Part 13: Interactive Disk Usage Analysis

Using ncdu for visual storage analysis

$ sudo apt install ncdu -y
$ ncdu /home
Part 14: System Logs Analysis

Monitoring and managing system and security logs

$ less /var/log/syslog
$ grep "sshd" /var/log/auth.log
Part 15: Task Scheduling

Managing automated tasks with Cron

$ crontab -e
# */2 * * * * /usr/bin/uptime
$ crontab -l
