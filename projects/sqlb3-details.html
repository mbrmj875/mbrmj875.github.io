Практические упражнения

00

Упражнение 00: Движение влево, движение вправо

Написать SQL запрос, который возвращает список пиццерий с соответствующим рейтингом, которые не были посещены людьми.

-- Решение:
SELECT pz.name, pz.rating
FROM pizzeria pz
LEFT JOIN person_visits pv ON pz.id = pv.pizzeria_id
WHERE pv.pizzeria_id IS NULL;
01

Упражнение 01: Поиск пропущенных данных

Найти пропущенные дни с 1 по 10 января 2022 года для посещений людей с идентификаторами 1 или 2.

-- Решение:
SELECT generate_series('2022-01-01'::date,
    '2022-01-10'::date, '1 day'::interval)::date AS missing_date
EXCEPT
SELECT visit_date
FROM person_visits
WHERE person_id IN (1, 2)
ORDER BY missing_date;
02

Упражнение 02: FULL означает 'полностью заполнен'

Вернуть полный список имен людей и пиццерий с 1 по 3 января 2022 года.

-- Решение:
SELECT
  COALESCE(p.name, '-') AS person_name,
  pv.visit_date,
  COALESCE(pz.name, '-') AS pizzeria_name
FROM person p
FULL JOIN person_visits pv ON p.id = pv.person_id
FULL JOIN pizzeria pz ON pv.pizzeria_id = pz.id
WHERE pv.visit_date BETWEEN '2022-01-01' AND '2022-01-03'
  OR pv.visit_date IS NULL
ORDER BY person_name, visit_date, pizzeria_name;
03

Упражнение 03: Преобразование в CTE

Переписать SQL из Упражнения 01 с использованием CTE (Common Table Expression).

-- Решение:
WITH date_series AS (
    SELECT generate_series('2022-01-01'::date,
        '2022-01-10'::date, '1 day'::interval)::date AS day
)
SELECT day AS missing_date
FROM date_series
LEFT JOIN person_visits pv ON date_series.day = pv.visit_date
    AND pv.person_id IN (1, 2)
WHERE pv.visit_date IS NULL
ORDER BY missing_date;
04

Упражнение 04: Поиск любимых пицц

Найти информацию о всех возможных названиях пиццерий и ценах на грибную или пепперони пиццу.

-- Решение:
SELECT m.pizza_name, pz.name AS pizzeria_name, m.price
FROM menu m
JOIN pizzeria pz ON m.pizzeria_id = pz.id
WHERE m.pizza_name IN ('mushroom pizza', 'pepperoni pizza')
ORDER BY m.pizza_name, pz.name;
05

Упражнение 05: Исследование данных о людях

Найти имена всех женщин старше 25 лет.

-- Решение:
SELECT name
FROM person
WHERE gender = 'female' AND age > 25
ORDER BY name;
06

Упражнение 06: Любимые пиццы Дениса и Анны

Найти все названия пицц (и соответствующие названия пиццерий), заказанные Денисом или Анной.

-- Решение:
SELECT m.pizza_name, pz.name AS pizzeria_name
FROM person_order po
JOIN person p ON po.person_id = p.id
JOIN menu m ON po.menu_id = m.id
JOIN pizzeria pz ON m.pizzeria_id = pz.id
WHERE p.name IN ('Denis', 'Anna')
ORDER BY m.pizza_name, pz.name;
07

Упражнение 07: Самая дешевая пиццерия для Дмитрия

Найти название пиццерии, которую Дмитрий посетил 8 января 2022 года и мог съесть пиццу дешевле 800 рублей.

-- Решение:
SELECT pz.name
FROM person_visits pv
JOIN pizzeria pz ON pv.pizzeria_id = pz.id
JOIN menu m ON pz.id = m.pizzeria_id
JOIN person p ON pv.person_id = p.id
WHERE p.name = 'Dmitriy'
  AND pv.visit_date = '2022-01-08'
  AND m.price < 800;
08

Упражнение 08: Продолжение исследования данных

Найти имена всех мужчин из Москвы или Самары, которые заказали пепперони или грибную пиццу.

-- Решение:
SELECT DISTINCT p.name
FROM person p
JOIN person_order po ON p.id = po.person_id
JOIN menu m ON po.menu_id = m.id
WHERE p.gender = 'male'
  AND p.address IN ('Moscow', 'Samara')
  AND m.pizza_name IN ('pepperoni pizza', 'mushroom pizza')
ORDER BY p.name DESC;
09

Упражнение 09: Кто любит сыр и пепперони?

Найти имена всех женщин, которые заказали и пепперони, и сырную пиццу.

-- Решение:
SELECT p.name
FROM person p
WHERE p.gender = 'female'
  AND EXISTS (
    SELECT 1 FROM person_order po
    JOIN menu m ON po.menu_id = m.id
    WHERE po.person_id = p.id
      AND m.pizza_name = 'pepperoni pizza'
  )
  AND EXISTS (
    SELECT 1 FROM person_order po
    JOIN menu m ON po.menu_id = m.id
    WHERE po.person_id = p.id
      AND m.pizza_name = 'cheese pizza'
  )
ORDER BY p.name;
10

Упражнение 10: Найти людей из одного города

Найти имена людей, которые живут по одному адресу (в одном городе).

-- Решение:
SELECT
  p1.name AS person_name1,
  p2.name AS person_name2,
  p1.address AS common_address
FROM person p1
JOIN person p2 ON p1.address = p2.address
WHERE p1.id > p2.id
ORDER BY person_name1, person_name2, common_address;
