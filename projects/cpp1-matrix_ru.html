Обзор проекта

Проект s21_matrix+ представляет собой объектно-ориентированную реализацию библиотеки для работы с матрицами на языке C++. В отличие от процедурного подока в оригинальном проекте s21_matrix, здесь используется современный объектно-ориентированный подход с перегрузкой операторов, конструкторами, деструкторами и обработкой исключений.

Примеры использования

Пример 1: Естественный синтаксис операций
#include "s21_matrix_oop.h" int main() {
  // Создание матриц   S21Matrix A(2, 2);   S21Matrix B(2, 2);   S21Matrix C(2, 2);
  // Естественный синтаксис операций   C = A + B;   C = A * B;   C += A;   if (A == B) {     // ...   }   // Индексация   double value = A(1, 1);   A(1, 1) = 42.0; }
Пример 2: Обработка исключений
void matrix_operations() {
  try {     S21Matrix A(2, 3);     S21Matrix B(4, 2);     S21Matrix C = A * B; // Ошибка: несовместимые размеры   } catch (const std::exception& e) {     std::cout << "Ошибка: " << e.what() << std::endl;   }   try {     S21Matrix D(2, 2);     double det = D.Determinant(); // OK     S21Matrix rect(2, 3);     double bad_det = rect.Determinant(); // Ошибка: не квадратная   } catch (const std::exception& e) {     std::cout << "Ошибка: " << e.what() << std::endl;   } }
