Project Overview

The C7_BrickGame_v1.0 project is a complete implementation of the classic Tetris game using C language and the NCurses library for terminal interface. The project features advanced modular architecture that separates game logic from user interface, with implementation of a finite state machine for managing game flow.

Advanced Scoring Algorithm
// Score calculation system void update_score(GameState_t *gs, int lines_cleared) {   switch (lines_cleared) {     case 1: gs->score += 100; break;     case 2: gs->score += 300; break;     case 3: gs->score += 700; break;     case 4: gs->score += 1500; break;   }   // Update level every 600 points   gs->level = (gs->score / 600) + 1;   gs->speed = 1000 - (gs->level * 100); }
